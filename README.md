# Bangle.js Pixel Graphics Application

A lightweight pixel-based graphics application developed for the Bangle.js smartwatch using the Espruino JavaScript framework.  
The project demonstrates how graphical elements can be rendered and controlled on a resource-constrained wearable embedded device.

---

## Project Overview

This project explores graphical rendering on the **Bangle.js smartwatch**, a programmable wearable platform based on the **Espruino JavaScript environment**.  

The application creates a pixel-level graphical display where individual pixels are manipulated to generate visual patterns and interactive display elements.

The goal of the project was to understand:

- Graphics rendering on embedded wearable devices
- Memory-efficient pixel manipulation
- Lightweight JavaScript execution in constrained environments
- Event-driven display updates on embedded hardware

This work was implemented as part of an academic exploration of **embedded and ubiquitous computing systems**.

## Software Environment

**Language:** JavaScript  

**Framework:** Espruino

## How It Works

The application controls the Bangle.js display at the pixel level.

The main implementation performs the following steps:

1. Initializes the display environment
2. Defines pixel coordinates for drawing
3. Updates graphical elements dynamically
4. Renders visual output on the watch screen

The code is designed to remain **lightweight and efficient**, which is important for embedded wearable devices with limited computational resources.

---

## Running the Project

To run the project:

1. Connect the **Bangle.js smartwatch**
2. Open the **Espruino Web IDE**
3. Upload the script:
src/pixel_app.js


4. Execute the program on the device

The display will render the pixel graphics directly on the watch screen.
